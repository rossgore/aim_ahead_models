Usage
bash
python3 medicaid_policy_simulator.py \
  --population output/synthetic_population.csv \
  --policy policies/income_tightening.csv \
  --policy policies/admin_churn.csv \
  --output-dir output
Or baseline only (no policies):

bash
python3 medicaid_policy_simulator.py \
  --population output/synthetic_population.csv \
  --output-dir output
